# Dolraj Neupane — Portfolio

Personal portfolio website for Dolraj Neupane, backend developer specializing in C++, C#, Python, JavaScript, PHP, and Laravel.

## Stack

- Vanilla HTML5, CSS3, JavaScript (no frameworks)
- Google Fonts: JetBrains Mono + Inter
- Fully static — no build step required

## Features

- **Dark / Light mode** with localStorage persistence
- **5 color themes**: Default, Cyberpunk, Ocean, Forest, Sunset
- **Mobile-first** responsive layout with fixed bottom navigation
- **Scroll reveal** animations (IntersectionObserver)
- **Reduced motion** support via `prefers-reduced-motion`

## Structure

```
portfolio/
├── index.html
├── css/
│   └── styles.css
├── js/
│   └── main.js
├── README.md
└── LICENSE
```

## Running Locally

Open `index.html` directly in any modern browser — no server needed.

Or use a simple local server:

```bash
npx serve .
# or
python3 -m http.server 8080
```
